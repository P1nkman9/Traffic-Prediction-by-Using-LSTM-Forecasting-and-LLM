# Traffic-Prediction-by-Using-LSTM-Forecasting-and-LLM
ELEC0149 Final Report Group 17

This project uses an LSTM model to predict traffic SPI and integrates Gemini 2.5 Flash for natural-language explanations. 

To run the cw.ipynb in Google Colab: 

1. Upload vel_metr_la.csv to Google Drive and replace the file path in the code.

2. Set your GEMINI_API_KEY in the code.

3. Execute all cells to train the model and get LLM-based insights.
